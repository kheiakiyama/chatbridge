Chat Bridge

ChatBridge は異なるチャットサービス間でテキストメッセージをやりとりするサービスです
現在 Slack, Skype, WebChat に対応しています



How to use

  1. 左のボタンからあなたが使うチャットサービスのBotを登録します
    WebChat は左のフォームを使うか、以下のコードを埋め込んでください 
    <iframe src='https://webchat.botframework.com/embed/chatbridge?s=k2nED_Y4Gfg.cwA.NDc.c7jRR7xdtm7UHotygCbUGjU5EDX8isWIkrPf-55D18k'></iframe>
  2. 登録したチャットサービス上で 
    create bridge
    コマンドを @chatbridge に送ってください
    @chatbridge はあなたに 
    open bridge {GUID}
    のような返信を送ります
  3. {GUID} をチャットする相手に教えてあげてください
    相手が 
    open bridge {GUID}
    コマンドを @chatbridge に送ると、あなたと接続が確立します
  4. 準備は以上です
    close bridge
    するまでの間、 @chatbridge に送ったメッセージは接続中の相手全員に送信されます
